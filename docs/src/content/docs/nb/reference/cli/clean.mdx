Fjern CLI-hurtigbufferkatalogen.

```bash
archgate clean
```

Fjerner `~/.archgate/`, som lagrer hurtigbufrede data som tidsstempler for oppdateringssjekker. Trygt å kjøre når som helst -- katalogen gjenskapes automatisk ved behov.

## Eksempel

```bash
archgate clean
```

```
/home/user/.archgate cleaned up
```
